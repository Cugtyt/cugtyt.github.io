题目描述

输入一个链表，输出该链表中倒数第k个结点。

``` c++
ListNode* FindKthToTail(ListNode* pListHead, unsigned int k) {
    int count = 0;
    auto head = pListHead;
    while (pListHead) {
        pListHead = pListHead->next;
        ++count;
    }
    if (count < k) {
        return NULL;
    }
    int step = count - k;
    while (step--) {
        head = head->next;
    }
    return head;
}
```
