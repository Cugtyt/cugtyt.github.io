题目描述

数组中有一个数字出现的次数超过数组长度的一半，请找出这个数字。例如输入一个长度为9的数组{1,2,3,2,2,2,5,4,2}。由于数字2在数组中出现了5次，超过数组长度的一半，因此输出2。如果不存在则输出0。

``` c++
int MoreThanHalfNum_Solution(vector<int> numbers) {
    if (numbers.size() < 1) {
        return 0;
    }
    
    int num = numbers[0];
    int count = 1;
    for (auto n : numbers) {
        if (n == num) {
            ++count;
        } else {
            --count;
            if (count == 0) {
                num = n;
                count = 1;
            }
        }
    }
    count = 0;
    for (auto n: numbers) {
        count += n == num;
    }
    return count > numbers.size() / 2 ? num : 0;
}
```
