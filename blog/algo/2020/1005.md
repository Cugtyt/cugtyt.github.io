实现 int sqrt(int x) 函数。

计算并返回 x 的平方根，其中 x 是非负整数。

由于返回类型是整数，结果只保留整数的部分，小数部分将被舍去。

示例 1:
```
输入: 4
输出: 2
```

示例 2:
```
输入: 8
输出: 2
说明: 8 的平方根是 2.82842..., 
     由于返回类型是整数，小数部分将被舍去。
```

``` python
class Solution:
    def mySqrt(self, x: int) -> int:
        if x in [0, 1]: return x
        cur, prev = 2, 0
        while int(cur) != prev:
            prev = int(cur)
            cur = (x / cur + cur) / 2

        return prev
```
